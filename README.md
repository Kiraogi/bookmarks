
# Bookmarks

Bookmarks - это веб-приложение для управления и обмена закладками. Проект предоставляет пользователям возможность сохранять, организовывать и делиться закладками на различные веб-ресурсы.

## Основные возможности

- **Управление закладками**: пользователи могут добавлять, удалять и редактировать закладки.
- **Социальные функции**: возможность подписываться на других пользователей и видеть их закладки, также есть история изменений ("Что нового?").
- **Лайки и комментарии**: пользователи могут ставить лайки и оставлять комментарии к закладкам.
- **Рейтинг закладок**: рейтинг закладок вычисляется на основе количества просмотров.

## Используемые технологии

- **Python**: основной язык программирования для серверной части приложения.
- **Django**: веб-фреймворк, используемый для разработки серверной части и управления базой данных.
- **HTML/CSS/JavaScript**: технологии для создания пользовательского интерфейса.
- **PostgreSQL**: реляционная база данных для хранения информации о пользователях и закладках.
- **Redis**: используется для кэширования и управления сессиями.
- **Docker**: контейнеризация приложения для упрощения развертывания и управления зависимостями.

## Установка и запуск

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Kiraogi/bookmarks.git
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv .venv
# Для Windows
.venv\Scripts\activate
# Для Unix или MacOS
source .venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Примените миграции базы данных:
```bash
python manage.py migrate
```

5. Подключить Redise через Docker:
```bash
docker pull redis
```

6. Запуск контейнера Docker для Redis:
```bash
docker run -it --rm --name redis -p 6379:6379 redis
```

7. Запуск клиента Redise:
```bash
docker exec -it redis sh
```

8. Запустите сервер разработки:
```bash
python manage.py runserver
```

Приложение будет доступно по адресу `http://localhost:8000` или `http://127.0.0.1:8000`

## Структура проекта

- `account/`: приложение для управления пользователями
- `actions/`: приложение для отслеживания действий пользователей
- `images/`: приложение для управления изображениями и закладками
- `bookmarks/`: основные настройки проекта
- `media/`: директория для хранения загруженных пользователями файлов
- `static/`: статические файлы (CSS, JavaScript)
- `templates/`: HTML шаблоны
